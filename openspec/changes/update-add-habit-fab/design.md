# Design: Floating Action Button

## Overview
The "Add Habit" primary action will be represented by a Floating Action Button (FAB) located in the bottom-right corner of the screen.

## Visual Design
- **Shape**: Circular
- **Icon**: "+" symbol (Plus icon)
- **Position**: Absolute position, bottom right.
- **Color**: Accent color (e.g., Primary/Brand color) or distinct from background.
- **Elevation**: Shadow for depth.

## Interaction
- **Tap**: Triggers the "Add Habit" flow (currently just a placeholder or navigation if defined).
- **Feedback**: Standard touch feedback (opacity or ripple).
